# Repository for Visual Analytics F26 at Aarhus University
This repo covers all four projects handed in during the course, each project within the folders features its own readme describing the specific project.

# Repo Structure
```
projects/
  └── cnn-classification # Using convolutional neural networks to classify LEGO bricks
  └── face-detection # Using a face detection algorithm to count and plot how many faces appear in newspapers through history
  └── image-classification-benchmarks # Comparing simple logistic regression against neural network image classification on Cifar10
  └── image-search # Comparing simple color histograms to VGG as a feature extraction mechanism for an image search algorithm

README.md
```
